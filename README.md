Project name: Website Health Checker

Description:
Pre-determined website links are indicated in dictionary. This project automatically,
based on a set time interval (Github action), runs the Python script to check each of
the website's status - whether they are up (status code = 200) or currently down.

If any of the website is down, the user is notified through a Telegram bot that was
setup to handle this scenario.
